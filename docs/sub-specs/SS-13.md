# SS-13: Signal 5 — Event Risk (Modifier)

**Master Spec Sections:** §4.5
**Dependencies:** SS-01, SS-04, SS-05, SS-07
**Estimated LOC:** ~150

## Scope

Computes Signal 5 (Event Risk) — a non-directional modifier in [0, 1.0] measuring probability of sudden price moves. Takes maximum of 5 risk components: options expiry, liquidation cascade, gamma flip proximity, implied volatility (DVol), and macro event proximity. When > 0.8 (IMMUTABLE), system outputs "STAY OUT". Does NOT compute directional signals or trade plans.

## Key Requirements

1. Component 1 — Options Expiry: Expiry < 24h AND notional > threshold → 0.8; < 48h AND > threshold → 0.4; else 0.0.
2. Component 2 — Liquidation Cascade: total_liq > extreme → 0.9; > high → 0.5; > medium → 0.2; else 0.0.
3. Component 3 — Gamma Flip Proximity: |distance| < close_pct → 0.7; < near_pct → 0.3; else 0.0.
4. Component 4 — DVol: > extreme → 0.8; > high → 0.4; > moderate → 0.1; else 0.0.
5. Component 5 — Macro Event Proximity: Tier 1 < 2h → 0.95; < 6h → 0.70; < 24h → 0.40; Tier 2 < 2h → 0.60; < 6h → 0.30; Tier 3 < 2h → 0.30; else 0.0.
6. Final = max(all components). Range [0, 1.0].
7. STAY OUT if event_risk > 0.8 (IMMUTABLE threshold from config).

## Acceptance Criteria

- [ ] Options expiry risk scores correctly by notional and time
- [ ] Liquidation cascade maps to correct risk tiers
- [ ] Gamma flip proximity scores correctly
- [ ] DVol maps to correct risk tiers
- [ ] Macro event proximity scores by tier and time
- [ ] Final risk = max of all components
- [ ] Range is [0, 1.0]
- [ ] All thresholds from config
