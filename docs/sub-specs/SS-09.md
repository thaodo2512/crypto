## Scope

Computes Composite Signal 1 (Spot Flow) ‚Äî a directional score in [-1, +1] answering "Where is real money actually flowing?" Combines CVD z-score with divergence detection, whale trade ratio, and order book imbalance, adjusted by a volume multiplier. This sub-spec does NOT collect raw data (SS-02), compute CVD z-scores (SS-07), or combine with other signals (SS-15).

## Key Requirements

1. `compute_spot_flow(db_path, config)` ‚Äî Orchestrator that computes final signal score.
2. Component 1 ‚Äî CVD Analysis (weight from config `spot_flow.cvd_weight`): Uses `compute_cvd_zscore()` from SS-07, clipped to [-1, +1]. If divergent (from `compute_cvd_divergence()`), amplify by `cvd_divergence_amplifier`. If 4h contradicts 24h (from `compute_cvd_4h_confirmation()`), reduce by `cvd_4h_contradiction_factor`.
3. Component 2 ‚Äî Whale Trade Ratio (weight from config `spot_flow.whale_weight`): `score = (whale_ratio - 0.5) √ó 2`, range [-1, +1].
4. Component 3 ‚Äî Order Book Imbalance (weight from config `spot_flow.orderbook_weight`): Score = imbalance value (already [-1, +1]). Anti-spoof filter: if `|imbalance| > anti_spoof_threshold`, reduce to `anti_spoof_factor`.
5. Volume Multiplier: `volume_ratio > volume_high_threshold` ‚Üí multiply by `volume_high_multiplier`; `volume_ratio < volume_low_threshold` ‚Üí multiply by `volume_low_multiplier`; otherwise linear interpolation `volume_base + volume_ratio √ó volume_slope`.
6. Final = `clip((C1√ów1 + C2√ów2 + C3√ów3) √ó volume_multiplier, -1, +1)`.
7. On any data failure, return score 0.0 (neutral).
