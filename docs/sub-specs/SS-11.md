# SS-11: Signal 3 — Options Structure

**Master Spec Sections:** §4.3
**Dependencies:** SS-01, SS-07
**Estimated LOC:** ~150

## Scope

Computes Composite Signal 3 (Options Structure) — a directional score in [-1, +1] answering "Are market maker hedge flows supporting or opposing the current price direction?" Combines gamma flip distance, net GEX level z-score, IV skew (contrarian), and max pain gravity. Uses pre-computed values from SS-07. Does NOT compute Greeks/GEX (SS-07) or combine with other signals (SS-15).

## Key Requirements

1. Component 1 — Gamma Flip Distance (weight `options_structure.gamma_flip_weight`): `flip_distance_pct = (spot - gamma_flip) / spot × 100`. Maps distance to score per §4.3 thresholds.
2. Component 2 — Net GEX Level (weight `options_structure.net_gex_weight`): Z-score of total_net_gex against 30-day history. Maps z-score to score per §4.3.
3. Component 3 — IV Skew (weight `options_structure.iv_skew_weight`): Contrarian — positive skew (put demand) → bullish. Maps skew to score per §4.3.
4. Component 4 — Max Pain Gravity (weight `options_structure.max_pain_weight`): `mp_distance_pct = (spot - max_pain) / spot × 100`. Maps distance to score per §4.3.
5. Final = `clip(C1×w1 + C2×w2 + C3×w3 + C4×w4, -1, +1)`.
