# SS-20: Evaluation & Performance Tracking

**Master Spec Sections:** §16
**Dependencies:** SS-15, SS-18
**Estimated LOC:** ~400

## Scope

Tracks signal outcomes at 4h/12h/24h/48h horizons. Computes win rate, component accuracy (correlation), regime-specific performance, and generates weekly/monthly reports. Includes background outcome tracker that fills in future prices. Does NOT modify signals or weights automatically — all changes require human approval.

```
custom/evaluation/
└── tracker.py

tests/
└── test_tracker.py
```

## Key Requirements

1. `OutcomeTracker` class fills in actual prices for past signals at 4h/12h/24h/48h horizons.
2. `fill_outcomes(db_path)` finds signals with missing outcome data and fills from price history.
3. `evaluate_correctness(bias, price_at, price_after)` determines if prediction was correct.
4. `compute_win_rate(db_path, days)` computes rolling win rate over N days.
5. `compute_component_accuracy(db_path, days)` computes per-signal correlation with outcomes.
6. `compute_regime_accuracy(db_path, days)` computes win rate segmented by regime.
7. `generate_weekly_report(db_path, config)` generates weekly summary string.
8. `generate_monthly_report(db_path, config)` generates monthly summary string.
9. Primary horizon: 24h. Neutral correct if |change| < 1%.
10. Minimum sample sizes enforced (30 overall, 15 per regime).

## Acceptance Criteria

- [ ] AC1: Outcome tracker fills prices at 4h/12h/24h/48h horizons
- [ ] AC2: Correctness: LONG correct if price went up, SHORT correct if down
- [ ] AC3: Neutral correct if |change| < 1%
- [ ] AC4: Win rate computed over rolling 30-day window
- [ ] AC5: Component accuracy computed per signal
- [ ] AC6: Accuracy segmented by regime
- [ ] AC7: Weekly report includes win count, loss count, net R
- [ ] AC8: Returns "insufficient data" when < 30 samples
- [ ] AC9: All horizons from config

## Integration Points

- **Inputs:** `signals` table, `spot_price` table, config
- **Outputs:** Performance reports via SS-18 (Telegram), metrics for SS-23 (Dashboard)
